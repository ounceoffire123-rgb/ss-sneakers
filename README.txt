SS Sneakers — Site Catálogo (Pronto)

✅ Tudo pronto para você só adicionar itens, preços, fotos e descrição.

ARQUIVOS IMPORTANTES
1) products.json  -> aqui você adiciona/edita produtos
2) config.json    -> aqui você coloca o link do Instagram da loja (quando criar) e o WhatsApp (se mudar)
3) assets/images/ -> aqui você sobe as fotos (e o logo já está dentro)

COMO ADICIONAR PRODUTO (products.json)
- Duplique um item e altere:
  - id (número único)
  - brand (uma das marcas)
  - name (modelo)
  - price (ex: 449.90)
  - sizes (ex: "37 ao 44")
  - image (ex: "assets/images/air-force-1.jpg")
  - description (texto curto)

DICA
- Suba as fotos na pasta assets/images
- Use nomes simples: air-force-1.jpg, ultraboost.jpg, nb-9060.jpg

PUBLICAR NO VERCEL (via GitHub)
- Suba todos os arquivos para o repositório
- No Vercel, importe o repositório e clique Deploy
- Toda alteração no GitHub atualiza o site automaticamente

OBS
- O site tem Busca + Filtro por marca + seções por marca.
- Venda via WhatsApp/Instagram.
